ℹ️ О чекере читеров

📋 Назначение

Проверка Steam-профилей на наличие VAC-банов, игровых банов и других ограничений.

🔍 Используемые сервисы

Steam Web API (GetPlayerBans, GetPlayerSummaries, ResolveVanityURL)

📊 Проверяемые данные

  • VAC-баны (количество)
  • Игровые баны
  • Коммьюнити-бан
  • Торговый бан
  • Дни с последнего бана

🔗 Поддерживаемые форматы ссылок

  • https://steamcommunity.com/id/USERNAME
  • https://steamcommunity.com/profiles/STEAMID64

🤖 Discord-команды

  • .!. <steam_url> — проверить один профиль
  • .!. vac N — показать последние N забаненных (1-100)
  • .!. vac-help — справка

🔎 Сторонние ресурсы для ручной проверки

  • steamid.io — информация о профиле и SteamID
  • vacbanned.com — все виды банов
  • convars.com — статистика CS2 и баны
