Камера и вид

1. Что такое HLAE и зачем он нужен

HLAE (Half-Life Advanced Effects) — инструмент для создания контента в CS2:

  • Запись видео без потери кадров (frame-perfect)
  • Плавные кинематографические пути камеры по демке
  • Свободная камера на паузе
  • Убирание HUD, фильтрация уведомлений об убийствах
  • Изменение FOV, скайбокса, эффектов
  • Запись отдельных слоёв (фон, игроки, оружие)
⚠️ Важно: HLAE запускает CS2 в режиме -insecure. Никогда не заходи на VAC-серверы через HLAE — бан. Только демки и LAN.

2. Установка и запуск CS2 через HLAE

Установка

  1. Скачать HLAE_Setup.exe с GitHub Releases
  2. Установить — путь не должен содержать кириллицу (например: C:\HLAE)
  3. Опционально — установить FFmpeg через тот же установщик

Запуск CS2

Открыть HLAE.exe → Menu → File → Launch CS2

ПолеЧто указать
Game...\cs2.exe
Moviemaking config parent folderПапка без кириллицы, например C:\HLAE\cfg
Graphic ResolutionНужное разрешение (1920x1080)
Custom command line options-console (опционально)

Нажать Launch. В консоли CS2 должно появиться сообщение от advancedfx.org.

Игровая логика и поведение

3. Загрузка и управление демкой

Загрузить демку

demoui // открыть интерфейс демок (или Shift+F2) demo_gototick 0 // перейти в начало демки

Управление скоростью

host_timescale 1 // нормальная скорость host_timescale 0.25 // четверть скорости (для campaths) host_timescale 0.1 // очень медленно host_timescale 0 // пауза

Прыжки по тикам

demo_gototick 12345 // перейти на конкретный тик mirv_skip time -5 // откатиться на 5 секунд назад mirv_skip time 10 // прыгнуть вперёд на 10 секунд mirv_skip tick to 10000 // перейти к тику 10000 mirv_skip time to 173.5 // перейти ко времени 173.5 сек

Рекомендуемые бинды

bind "F1" "demo_togglepause" bind "F2" "host_timescale 0.25" bind "F3" "host_timescale 1" bind "PGUP" "mirv_skip time 5" bind "PGDN" "mirv_skip time -5"

4. Свободная камера — mirv_input

mirv_input camera // включить свободную камеру mirv_input off // выключить, вернуть обычный режим
💡 Для POV-демок (от первого лица) сначала переключись в thirdperson:
sv_cheats 1 → thirdperson → mirv_input camera

Чувствительность

mirv_input cfg msens 1.0 // чувствительность мыши mirv_input cfg ksens 200 // скорость клавиатуры (WASD) mirv_input cfg mouseMoveSupport 1 // управление кнопками мыши

Некоторые команды для камеры

5. Плавные камера-пути (Campaths)

Campath — анимированный путь камеры. HLAE плавно интерполирует между расставленными точками.

Принцип работы

  1. Расставляешь кейфреймы (минимум 4) в нужных местах/моментах
  2. HLAE строит плавный путь между ними
  3. При воспроизведении камера сама летит по пути

Подготовка

host_timescale 0.25 // замедли демку mirv_input camera // включи свободную камеру mirv_campath draw enabled 1 // включи визуализацию пути

Расстановка кейфреймов

bind "E" "mirv_campath add" // добавить кейфрейм
⚠️ Нужно минимум 4 кейфрейма. Меньше — campath не включится!

Включение campath

mirv_campath enable 1 // включить путь камеры mirv_campath enable 0 // выключить mirv_campath print // список всех кейфреймов

Все команды campath

КомандаОписание

mirv_campath save mypathСохранить путь в файл

mirv_campath edit startСдвинуть путь к текущему времени
mirv_campath edit duration 10Задать длительность (сек)
mirv_campath setduration 10Растянуть путь до 10 сек

Интерполяция

mirv_campath edit interp position linear // линейное движение mirv_campath edit interp position cubic // кубическое (по умолчанию) mirv_campath edit interp rotation sLinear // линейный поворот mirv_campath edit interp fov linear // линейное изменение FOV

Бинды для campath

bind "E" "mirv_campath add" bind "R" "mirv_campath enable 1" bind "T" "mirv_campath enable 0" bind "G" "mirv_campath clear" bind "F4" "mirv_campath draw enabled 1" bind "F5" "mirv_campath draw enabled 0" bind "F6" "mirv_campath print"

Советы

  • Расставляй точки в слоу-мо (host_timescale 0.1–0.25), не на паузе
  • Между кейфреймами минимум 1–2 секунды реального времени
  • Используй mirv_campath draw enabled 1 чтобы видеть путь
  • Сохраняй пути через mirv_campath save — можно редактировать XML

Экспорт камеры в файл

mirv_camio export start mycamera.xml // начать экспорт mirv_camio export end // остановить

Альтернатива: mirv_streams record cam enabled 1 — экспорт автоматически при записи.

7. Полезные команды для чистой картинки

Убрать HUD

cl_drawhud 0 // скрыть HUD cl_drawhud 1 // вернуть cl_drawonlydeathnotices 1 // только прицел + killfeed

Флэшбэнг

mirv_endofmatch 1 // убрать экран окончания

Скрытые конвары

mirv_cvar_unhide_all // все скрытые команды mirv_cvar_unlock_sv_cheats // обойти sv_cheats

Дым

Скайбокс

mirv_sky enabled 1 // включить управление mirv_sky color r 0.1 g 0.2 b 0.5 // цвет неба (RGB 0.0–1.0) mirv_sky clouds draw 0 // скрыть облака mirv_sky clouds color r 255 g 200 b 150 // цвет облаков mirv_sky material "materials/skybox/cs_italy_s2_skybox_2.vmat_c" mirv_sky material default // вернуть стандартный

Цвета свечения (mirv_colors)

10. Системные команды

Список сущностей

mirv_listentities // все сущности mirv_listentities isPlayer=1 // только игроки mirv_listentities sort=distance limit=10 // 10 ближайших

Исправления

mirv_fix time 1 // исправить время Panorama mirv_fix animations 1 // исправить плавность анимаций

Загрузка DLL

mirv_loadlibrary "C:/path/to/library.dll"

Приоритет камеры

  1. mirv_campath (высший)
  2. mirv_camimport
  3. mirv_fov
  4. mirv_input camera (низший)

Скрипты и команды

Авто-выполнение команд

mirv_cmd add time 45.5 "cl_drawhud 0" mirv_cmd add tick 12000 "mirv_campath enable 1" mirv_cmd clear mirv_cmd save "script.xml" mirv_cmd load "script.xml"

Анимация по кривым

mirv_cmd clear mirv_cmd addCurves tick 1000 4000 - interp=cubic space=abs 1000 1 1300 2 1600 0.5 1900 2 2100 0.5 2500 2 2800 1 3100 2 3400 0.5 3700 2 4000 1 - interp=linear 0 0 1 100 -- "mirv_time drive {0}; echo {1}% finished: driving: {0}" mirv_cmd addAtTick 4000 "mirv_time drive default"

{0} — кубическая интерполяция (мин. 4 точки). {1} — линейная (мин. 2 точки).

12. Полный воркфлоу

Этап 1 — Подготовка

  1. Запустить CS2 через HLAE → File → Launch CS2
  2. Дождаться сообщения от advancedfx.org
  3. Выполнить exec hlae_movie
  4. Загрузить демку: demoui → Load

Этап 2 — Поиск моментов

  1. Смотришь демку на нормальной скорости (host_timescale 1)
  2. Нашёл момент — запоминаешь тик
  3. Откатываешься: mirv_skip time -10

Этап 3 — Создание campath

  1. Замедли: host_timescale 0.25
  2. Камера: mirv_input camera
  3. Визуализация: mirv_campath draw enabled 1
  4. Ставь точки: mirv_campath add (мин. 4)
  5. Проверь: mirv_campath print
  6. Включи: mirv_campath enable 1
  7. Откатись до начала

Этап 4 — Запись

  1. Визуализация: mirv_campath draw enabled 0
  2. HUD: cl_drawhud 0
  3. Запись: mirv_streams record start
  4. Запусти демку
  5. Стоп: mirv_streams record end
  6. Сброс: mirv_campath clear

Этап 5 — Монтаж

Gather files in Premiere / DaVinci Resolve / Vegas.

13. Частые ошибки

❌ Campath enabled but can not be evaluated

Причина: точек меньше 4.

Решение:

  • Используй mirv_campath add
  • Проверь: mirv_campath print
  • Точки в разное время демки
  • Между точками 1–2 секунды

❌ Видео не записывается

  • FFmpeg установлен
  • Путь без кириллицы
  • Права на запись

❌ HLAE не подцепился

  • Запускай через HLAE → File → Launch CS2
  • Путь к cs2.exe правильный
  • Путь к HLAE без кириллицы
