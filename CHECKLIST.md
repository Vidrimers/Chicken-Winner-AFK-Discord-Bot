# ✅ Чек-лист перед запуском

## 📋 Перед первым запуском

- [ ] Создан бэкап базы данных: `cp afkbot.db afkbot.db.backup`
- [ ] Проверен `.env` файл (все переменные на месте)
- [ ] Старый бот остановлен (нет процессов `node bot.js`)
- [ ] Установлены зависимости: `npm install`
- [ ] Прочитан [QUICKSTART.md](QUICKSTART.md)

## 🚀 Первый запуск

- [ ] Запущен новый бот: `npm start`
- [ ] Нет ошибок в консоли
- [ ] Discord клиент подключен
- [ ] Express сервер запущен на порту 3000
- [ ] Telegram бот инициализирован

## 🧪 Базовое тестирование

### Discord
- [ ] Зашел в голосовой канал - работает
- [ ] Команда `.!. stats` - работает
- [ ] Команда `.!. achievements` - работает
- [ ] AFK таймер (отключил микрофон) - работает

### Веб-панель
- [ ] Открывается http://localhost:3000
- [ ] Статистика отображается
- [ ] Настройки сохраняются
- [ ] Достижения показываются

### Telegram
- [ ] Команда `/start` - работает
- [ ] Кнопка "Кто в канале" - работает
- [ ] Уведомления приходят

## 📊 Проверка производительности

- [ ] Запросы к БД быстрые (< 10ms в логах)
- [ ] Веб-панель загружается быстро (< 1s)
- [ ] CPU использование нормальное (< 5%)
- [ ] Нет утечек памяти

## 📝 Полное тестирование (опционально)

## 🔄 Если что-то не работает

### Откат
```bash
# Остановить новый бот (Ctrl+C)
npm run start:old

# Восстановить БД
cp afkbot.db.backup afkbot.db
```

### Отладка
1. Проверь логи в консоли
2. Проверь `.env` файл
3. Читай [TESTING.md](.kiro/specs/code-refactoring/TESTING.md)
4. Сравни с bot.js (старая версия)
